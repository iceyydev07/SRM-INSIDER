#🚀 SRM Insider

A modern, student-focused web platform built to help SRM students explore **placements, internships, and campus life** — all in one place.

---

## 🌟 Overview

**SRM Insider** is a fully responsive and visually rich web application designed to provide students with real insights into their academic and career journey.

This project was developed as part of a **club recruitment challenge**, focusing on UI/UX design, frontend development, and real-world data integration.

---

## ✨ Features

* 🏠 **Landing Page** – Clean and modern UI with engaging hero section
* 🔐 **Authentication UI** – Login & Signup with validation (frontend only)
* 📊 **Dashboard** – Dynamic cards for:

  * Placements
  * Internships
  * Campus Life
* 🔍 **Search & Filter** – Easily find relevant content
* ⭐ **Bookmark System** – Save posts for later
* 🌙 **Dark/Light Mode** – Smooth theme switching
* 📄 **Content Page** – Detailed article view with clean typography
* ⚡ **Animations** – Smooth transitions using Framer Motion

---

## 🧠 Tech Stack

* ⚛️ **Next.js (React Framework)**
* 🎨 **Tailwind CSS**
* 🧩 **ShadCN UI**
* 🎞️ **Framer Motion**
* 🐍 **Python (Flask API)** *(for dataset integration)*
* 📊 **Kaggle Dataset Integration**

---

## 🔗 API Integration

Integrated a real dataset using a Python backend to simulate real-world data flow.

* Fetches placement-related data
* Displays dynamic content in dashboard
* Easily extendable to full backend
